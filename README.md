## 🛠️ Tecnologías Principales
* **React 18** (Functional Components & Hooks)
* **TypeScript** (Configuración estricta de tipos)
* **Context API** (Gestión de estado de temas)
* **Tailwind CSS + OKLCH** (Soporte nativo para temas de alto contraste)
* **ESLint Strict Mode** (Cero aserciones de tipo y tipado seguro)

---

## 🎨 Características del Sistema de Temas
El proyecto incluye una utilidad propietaria (`ThemeUtil`) que gestiona:
* Persistencia automática en `LocalStorage`.
* Detección de preferencias del sistema operativo.
* Resolución dinámica de temas múltiples (**Light**, **Dark** y el exclusivo tema **Emerald**).
* Integración nativa con sistemas de internacionalización (i18n).
